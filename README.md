# CelestiAI


All the requirements are on requeirements.txt file. You should create a enviroment and activate it
python3 -m venv venv_celestiai
source venv_celestiai/bin/activate

To run locally:
Create a secret key from groq website
with the enviroment activated, execute : export GROQ_API_KEY="your_secret_key"
run: streamlit run app.py
